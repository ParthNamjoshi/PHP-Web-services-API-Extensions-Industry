Project Overview

This project is developed as part of an assessment focused on **Web Services, APIs, and Extensions**.
It demonstrates how to design and implement a scalable and secure REST API system.

The API allows:

 Product management (CRUD operations)
User authentication using JWT
Order processing system
API testing using Postman
Standardized error handling

Concepts Covered

-REST API Architecture
-HTTP Methods (GET, POST, PUT, DELETE)
-HTTP Status Codes
-JSON Response Handling
-JWT Authentication
-API Testing (Postman)
-External API Integration (cURL)
-Rate Limiting & Error Handling

Authentication

-JWT (JSON Web Token) based authentication is implemented.
-Protected routes:

-Create Product
-Update Product
-Delete Product
-Orders APIs
